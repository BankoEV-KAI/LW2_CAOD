# LW2_CAOD - кольцевой список (ститическое распредление памяти)

**Задание на ЛР**

Реализация структуры **кольцевая очередь** на основе статического распределения памяти.

Функции, которые необходимо реализовать:

1. Проверка пустоты очереди;
2. Проверка заполненности очереди;
3. Добавление элемента в конец очереди;
4. Удаление элемента из начала очереди;
5. Вывод текущего состояния очереди на экран.
